A Place To Start

Since I'm in my second year, I've already learnt a few of the major programming languages including python and java. I personally prefer java but that can vary depending on people's perspective. The most famous language at this time is javascript so it won't hurt to know a thing or two about it.

It might be a bit intimidating to dive straight into complicated languages like java or C++, which is why I advise people to start of with languages with simpler syntax and flexibilty like python.
